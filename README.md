## 📦 Install

```bash
$ git clone https://github.com/uvite/semi-design-pro
$ cd semi-design-pro

# yarn
$ yarn install
$ yarn dev

# npm
$ npm install
$ npm run dev
```

## 🔨 Build
```bash
# yarn
$ yarn install
$ yarn build:dev

# npm
$ npm install
$ npm run build:dev
```
